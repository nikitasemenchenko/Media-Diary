**MediaDiary** is a Android application designed for personal tracking and cataloging of watched movies, series, anime and cartoons. The app allows users to search for new content, add it to their collections ("Want to watch," "Watching," "Watched"), assign personal ratings, and monitor detailed statistics of their media consumption.

## Installation

[ðŸ‘‰ **Download the APK**](https://drive.google.com/file/d/1AWXCY3RFFfCwKSbs5QQTIjLD_PCuWfwB/view?usp=sharing)  

**Or:**

1. Clone the repository:
   ```bash
    git clone https://github.com/nikitasemenchenko/Media-Diary.git
    ```
2.  **Get API Key:**
    The project works with Kinopoisk API for media data. You must get your own API key.

3.  **Add the Key to `local.properties`:**
    In the root directory of the project, create or edit the `local.properties` file and add the following line:
    ```properties
    KP_API_KEY="YOUR_KEY"
    ```

4.  Open the project in Android Studio.
5.  Sync Gradle and run the application on an emulator or a physical device.
